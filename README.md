# Blog
📝 Blog Application

A full-stack blog application that allows users to create, manage, and interact with posts through a secure and user-friendly system. This project demonstrates backend development best practices using clean architecture, authentication, and access control.

🚀 Features
🔐 User Authentication & Authorization
Secure login and registration
Users can only manage their own data
📝 Post Management
Create, read, update, and delete posts (CRUD)
Each post is linked to its owner
💬 Comment System
Add, edit, and delete comments
Comments are associated with posts and users
🛡️ Security
Protected endpoints
Data isolation between users
🧱 Clean Architecture
Layered structure (Controller, Service, Repository)
DTOs and Mappers for clean data handling
